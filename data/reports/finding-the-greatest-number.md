# Finding the Greatest Number

## Category

Miscellaneous

## Difficulty

Easy

## Concepts

* Conditional Statements
* Input/Output

## Code Logic

1. The program starts by importing the necessary Scanner class for input/output operations.
2. It then creates a Scanner object to read input from the user.
3. The program prompts the user to enter three numbers, which are stored in variables a, b, and c.
4. The program uses if-else statements to compare the numbers and determine which one is the greatest.
5. If a is greater than or equal to both b and c, it prints 'a is Greater'.
6. If b is greater than or equal to both a and c, it prints 'b is Greater'.
7. If none of the above conditions are true, it prints 'c is Greater', indicating that c is the greatest number.

## Time Complexity

O(1)

## Space Complexity

O(1)

Low

Beginner

## Explanation

This Java program takes three numbers as input from the user and determines which number is the greatest. It uses conditional statements to compare the numbers and prints the greatest number. The program is simple and does not involve any complex algorithms or data structures.
