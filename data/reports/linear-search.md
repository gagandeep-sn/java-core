# Linear Search

## Category

Arrays

## Difficulty

Easy

## Concepts

* Searching

## Code Logic

1. The search method takes an array and a target value as parameters.
2. It initializes a for loop to iterate through each element in the array.
3. Inside the loop, it checks if the current element matches the target value.
4. If a match is found, it immediately returns the index of the current element.
5. If the loop completes without finding a match, it returns -1 to indicate that the target is not in the array.
6. In the main method, an example array is created and the search method is called with the target value 8.
7. The result is printed to the console, which is the index of the target value in the array.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Medium

## Topic Mastery

Beginner

## Explanation

This program performs a linear search on an array to find the index of a target element. It takes an array and a target value as input, then iterates through the array to find the target. If the target is found, it returns the index; otherwise, it returns -1. The program demonstrates a basic searching algorithm.
