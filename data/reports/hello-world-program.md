# Hello World Program

## Category

Miscellaneous

## Difficulty

Easy

## Concepts

* OOP
* Input Output

## Code Logic

1. The program starts execution from the main method, which is the entry point of the Java program.
2. The main method takes an array of strings as arguments, but in this case, it's not used.
3. The System.out.println method is called to print the string 'Hello, World!' to the console.
4. The println method automatically appends a newline character at the end of the string, so the output is displayed on a new line.

## Time Complexity

O(1)

## Space Complexity

O(1)

Low

## Explanation

This Java program prints 'Hello, World!' to the console. It takes no input from the user and simply outputs a greeting message. The program uses the System.out.println method to display the message. The output is a simple string that is displayed on the console.
