# Find Maximum Element in an Array

## Category

Arrays

## Difficulty

Easy

## Concepts

* Arrays
* Loops
* Conditional Statements

## Code Logic

1. We initialize the maximum element (max) to the first element of the array (arr[0]).
2. We start a loop from the second element (index 1) to the last element of the array.
3. Inside the loop, we check if the current element (arr[i]) is greater than the current maximum (max).
4. If it is, we update the maximum (max) to the current element (arr[i]).
5. After the loop finishes, max will hold the maximum element in the array, which we then print to the console.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Medium

## Topic Mastery

Beginner

## Explanation

This program finds the maximum element in a given array. It takes an array of integers as input, iterates through each element, and keeps track of the maximum value found. The program outputs the maximum element in the array.
