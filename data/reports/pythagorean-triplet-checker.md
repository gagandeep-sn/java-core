# Pythagorean Triplet Checker

## Category

Math

## Difficulty

Easy

## Concepts

* Math
* Input Output

## Code Logic

1. The program prompts the user to input three integers a, b, and c.
2. It determines the two smaller numbers (small1 and small2) and the largest number (max) among the input numbers.
3. The program calls the triplet method with the two smaller numbers and the largest number as arguments.
4. Inside the triplet method, it calculates the square root of the sum of the squares of the two smaller numbers (d).
5. It checks if the calculated value (d) equals the largest number (c).
6. If they match, it prints that the numbers form a Pythagorean triplet; otherwise, it prints that they do not.

## Explanation

This program checks if three input numbers form a Pythagorean triplet. It takes three integers as input, calculates the square root of the sum of the squares of the two smaller numbers, and checks if it equals the third number. If they match, it prints that the numbers form a Pythagorean triplet; otherwise, it prints that they do not.
