# Volume of a Cone Calculator

## Category

Math

## Difficulty

Easy

## Concepts

* Mathematical Formulas
* User Input

## Code Logic

1. The program starts by importing the necessary Scanner class to read user input.
2. It then creates a new Scanner object to read input from the user.
3. The program prompts the user to enter the radius of the cone and stores the input in the variable 'r'.
4. Next, it prompts the user to enter the height of the cone and stores the input in the variable 'h'.
5. The program calculates the volume of the cone using the formula (1/3) * π * r^2 * h, where π is approximated as 3.14.
6. Finally, it prints the calculated volume to the console.

## Time Complexity

O(1)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This program calculates the volume of a cone given its radius and height. It takes user input for the radius and height, then uses the formula for the volume of a cone to calculate the result. The program outputs the calculated volume. The input is processed using simple mathematical operations.
