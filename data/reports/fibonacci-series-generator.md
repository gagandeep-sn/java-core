# Fibonacci Series Generator

## Category

Math

## Difficulty

Easy

## Concepts

* Loops
* Variables
* Basic Math Operations

## Code Logic

1. The program first prompts the user to enter a number, which represents the number of terms in the Fibonacci series to generate.
2. It initializes three variables: x and y to 0 and 1, which are the first two numbers in the Fibonacci series, and next to store the next number in the series.
3. The program then enters a for loop that runs n times, where n is the number of terms to generate.
4. Inside the loop, it prints out the current value of x, which is the next number in the Fibonacci series.
5. It then calculates the next number in the series by adding x and y, and stores it in the next variable.
6. The program then updates the values of x and y for the next iteration by swapping them and setting y to the next number in the series.
7. After the loop finishes, the program has printed out the first n numbers in the Fibonacci series.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Explanation

This program generates the Fibonacci series up to a given number of terms. It takes an integer input from the user, then prints out the corresponding number of Fibonacci numbers. The Fibonacci series is a sequence of numbers where each number is the sum of the two preceding ones.
