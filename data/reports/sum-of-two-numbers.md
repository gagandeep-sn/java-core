# Sum of Two Numbers

## Category

Math

## Difficulty

Easy

## Concepts

* Input Output
* Basic Operations

## Code Logic

1. The program starts by printing a message asking the user to enter the first number.
2. It uses a Scanner object to read the user's input and store it in the variable num1.
3. It then prints another message asking the user to enter the second number.
4. The user's input is read and stored in the variable num2.
5. The program calls the sum method, passing num1 and num2 as arguments.
6. Inside the sum method, the two numbers are added together and stored in the variable add.
7. The result is returned to the main method and printed to the console.

## Explanation

This Java program takes two numbers as input from the user, adds them together, and prints the result. It uses a Scanner object to get the input and a simple method to calculate the sum. The program demonstrates basic input/output operations and arithmetic calculations.
