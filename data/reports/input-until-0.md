# Input Until 0

## Category

Input Output

## Difficulty

Easy

## Concepts

* Input/Output
* Loops

## Code Logic

1. The program creates a Scanner object to read input from the user.
2. It initializes a variable 'sum' to 0 to store the sum of input numbers.
3. The program enters an infinite while loop, which will continue until the user enters 0.
4. Inside the loop, the program prompts the user to enter a number and reads the input using the Scanner object.
5. If the input number is 0, the loop breaks and the program exits the loop.
6. If the input number is not 0, the program adds the number to the 'sum' variable.
7. After the loop ends, the program prints the sum of all input numbers.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This Java program takes integers as input from the user until the user enters 0. It then calculates and prints the sum of all the input numbers. The program uses a while loop to continuously take input until the termination condition is met. The sum is calculated and printed after the loop ends.
