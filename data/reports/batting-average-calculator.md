# Batting Average Calculator

## Category

Input Output

## Difficulty

Easy

## Concepts

* Input/Output
* Basic Arithmetic Operations

## Code Logic

1. The program prompts the user to enter the total runs scored in a series.
2. It then prompts the user to enter the number of times the player is out.
3. If the player is not out (i.e., the number of times out is 0), the program sets the batting average to the total runs scored.
4. If the player is out, the program calculates the batting average by dividing the total runs scored by the number of times the player is out.
5. Finally, the program prints the calculated batting average.

## Time Complexity

O(1)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This Java program calculates the batting average of a player. It takes the total runs scored and the number of times the player is out as input. If the player is not out, the batting average is the total runs scored. Otherwise, it is the total runs scored divided by the number of times the player is out.
