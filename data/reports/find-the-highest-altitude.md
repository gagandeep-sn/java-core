# Find The Highest Altitude

## Category

Arrays

## Difficulty

Easy

## Concepts

* Arrays
* Loops

## Code Logic

1. Initialize variables to keep track of the current altitude and the highest altitude reached so far.
2. Iterate through the array of altitude gains.
3. For each gain, add it to the current altitude.
4. Check if the current altitude is higher than the highest altitude reached so far, and update the highest altitude if necessary.
5. After iterating through the entire array, the highest altitude reached is stored in the 'highestAltitude' variable.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This program calculates the highest altitude reached given an array of altitude gains. It iterates through the array, adding each gain to the current altitude and keeping track of the highest altitude reached. The program outputs the highest altitude. The input is an array of integers representing altitude gains.
