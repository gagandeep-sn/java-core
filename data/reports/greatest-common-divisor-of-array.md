# Greatest Common Divisor of Array

## Category

Arrays

## Difficulty

Easy

## Concepts

* Arrays
* Loops
* Conditional Statements

## Code Logic

1. Initialize variables to store the minimum and maximum numbers in the array.
2. Iterate through the array to find the minimum and maximum numbers.
3. Calculate the GCD of the minimum and maximum numbers by iterating from 1 to the minimum number and checking if both numbers are divisible by the current number.
4. Print the GCD of the maximum and minimum numbers.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Medium

## Topic Mastery

Beginner

## Explanation

This program finds the greatest common divisor (GCD) of the maximum and minimum numbers in an array. It iterates through the array to find the minimum and maximum values, then calculates the GCD of these two numbers. The GCD is the largest number that divides both the maximum and minimum numbers without leaving a remainder.
