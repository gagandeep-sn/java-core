# Area Calculator

## Category

Math

## Difficulty

Easy

## Concepts

* Input Output
* Math

## Code Logic

1. The program creates a Scanner object to read input from the user.
2. It prompts the user to enter the side length of an equilateral triangle.
3. The program reads the user's input as an integer and stores it in the variable 'a'.
4. It calculates the area of the equilateral triangle using the formula (Math.sqrt(3)/4)*a*a.
5. The program prints the calculated area to the console.
6. Finally, the program closes the Scanner object to prevent resource leaks.

## Time Complexity

O(1)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This Java program calculates the area of an equilateral triangle given the length of its side. The user is prompted to enter the side length, and the program uses the formula for the area of an equilateral triangle to calculate the result. The program then prints the calculated area.
