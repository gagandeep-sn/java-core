# InputAndOutputOf2DArray

## Category

Input Output

## Difficulty

Easy

## Concepts

* Scanner
* User Input

## Code Logic

1. Use nested loops to go through each row and column of the 2D array
2. The outer loop iterates over each row
3. The inner loop iterates over each column in the current row
4. Process or access each element at position [row][column]
5. Create a Scanner object to read user input from System.in
6. Prompt the user and read values using nextInt(), nextLine(), etc.
7. Perform the required computation with the input values
8. Display the result using System.out.println()
9. Close the scanner to release system resources

## Time Complexity

O(n * m)

## Space Complexity

O(n * m)

A input output program focusing on Scanner, User Input.
