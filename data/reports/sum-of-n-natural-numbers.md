# Sum of N Natural Numbers

## Category

Math

## Difficulty

Easy

## Concepts

* Loops
* Input/Output

## Code Logic

1. The program prompts the user to enter a number.
2. The input number is stored in the variable 'num'.
3. The 'sum' method is called with 'num' as an argument.
4. Inside the 'sum' method, a loop runs from 1 to 'num' (inclusive).
5. In each iteration of the loop, the current number is added to the 'sum' variable.
6. After the loop finishes, the 'sum' variable holds the total sum of the first 'num' natural numbers.
7. The result is returned from the 'sum' method and printed to the console.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This Java program calculates the sum of the first N natural numbers. It takes an integer input from the user, calculates the sum using a loop, and prints the result. The program uses a simple formula to calculate the sum, which is the sum of all numbers from 1 to N.
