# Input and Output of 2D Array

## Category

Arrays

## Difficulty

Easy

## Concepts

* 2D Arrays
* Input/Output

## Code Logic

1. We first declare a 3x3 2D array and a Scanner object to read input from the user.
2. We use two nested for loops to iterate over each element in the 2D array and read an integer from the user using the Scanner object.
3. We store each integer in the corresponding position in the 2D array.
4. After reading all the input, we use another nested for loop to iterate over each row in the 2D array.
5. For each row, we use the Arrays.toString() method to convert the row into a string and print it using System.out.println().

## Time Complexity

O(n)

## Space Complexity

O(n)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This program takes a 3x3 2D array as input from the user and then prints each row of the array. The user is prompted to enter 9 integers, which are stored in the 2D array. The program then prints each row of the array on a new line.
