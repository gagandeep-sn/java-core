# Finding Maximum and Minimum of Three Numbers

## Category

Miscellaneous

## Difficulty

Easy

## Concepts

* Input Output
* Variables
* Conditional Statements

## Code Logic

1. The program starts by importing the Scanner class for reading user input.
2. It prompts the user to enter three numbers, which are stored in variables a, b, and c.
3. The max method is called with the three numbers as arguments, which finds the maximum number by comparing each number with the current maximum.
4. The min method is called similarly to find the minimum number.
5. Inside the max and min methods, the program initializes the maximum or minimum variable with the first number and then checks if the other numbers are greater or lesser than the current maximum or minimum.
6. If a number is greater or lesser, it updates the maximum or minimum variable accordingly.
7. Finally, the program prints the maximum and minimum numbers found.

O(1)

O(1)

## Explanation

This Java program takes three numbers as input from the user and finds the maximum and minimum among them. It uses two separate methods, max and min, to calculate and print the results. The program demonstrates basic input/output operations and conditional statements.
