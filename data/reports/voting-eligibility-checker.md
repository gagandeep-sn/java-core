# Voting Eligibility Checker

## Category

Miscellaneous

## Difficulty

Easy

## Concepts

* Input Output
* Conditional Statements

## Code Logic

1. The program starts by importing the necessary Scanner class for user input.
2. It creates a Scanner object to read the user's input.
3. The program prompts the user to enter their age and stores it in the 'age' variable.
4. The 'eligibleOrNot' method checks if the age is 18 or above and returns a boolean result.
5. The main method calls 'eligibleOrNot' with the user's age and stores the result in the 'result' variable.
6. Based on the 'result', the program outputs whether the person is eligible or not eligible to vote.

## Time Complexity

O(1)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This program checks if a person is eligible to vote based on their age. It takes the age as input from the user, checks if it's 18 or above, and outputs whether the person is eligible or not. The program uses a simple conditional statement to make the decision. The output is a message indicating the person's eligibility status.
