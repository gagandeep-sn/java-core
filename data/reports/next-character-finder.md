# Next Character Finder

## Category

Miscellaneous

## Difficulty

Easy

## Concepts

* Input Output
* Type Casting

## Code Logic

1. The program prompts the user to enter a character and uses a Scanner object to read the input.
2. The input character is converted to its corresponding ASCII value using type casting.
3. The ASCII value is incremented by 1 to get the ASCII value of the next character.
4. The incremented ASCII value is converted back to a character using type casting.
5. The next character is printed to the console.

## Time Complexity

O(1)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This Java program takes a single character as input from the user, finds the next character in the ASCII table, and prints it. The program uses type casting to convert the character to its corresponding ASCII value, increments it to get the next character's ASCII value, and then converts it back to a character. The program assumes that the input character is a single character. The output is the next character in the ASCII table.
