# Calculate Discount

## Category

Miscellaneous

## Difficulty

Easy

## Concepts

* Input Output
* Basic Operations

## Code Logic

1. The program starts by importing the Scanner class to read input from the user.
2. It creates a Scanner object to read input from the console.
3. The program prompts the user to enter the full price of an item and stores it in the variable 'Price'.
4. It then prompts the user to enter the discount percentage and stores it in the variable 'Dis'.
5. The program calculates the discount amount by multiplying the full price with the discount percentage and dividing by 100.
6. It then calculates the discounted price by subtracting the discount amount from the full price.
7. Finally, the program outputs the discounted price to the console.

## Time Complexity

O(1)

## Space Complexity

O(1)

Low

## Explanation

This program calculates the discounted price of an item. It takes the full price and discount percentage as input from the user, calculates the discount amount, and then subtracts it from the full price to get the discounted price. The program outputs the discounted price. It is a simple program that demonstrates basic input/output operations and arithmetic calculations.
