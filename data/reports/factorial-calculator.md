# Factorial Calculator

## Category

Math

## Difficulty

Easy

## Concepts

* Loops
* Input/Output

## Code Logic

1. The program creates a Scanner object to read the user's input.
2. It prompts the user to enter a number and stores the input in the variable 'n'.
3. The variable 'fac' is initialized to 1, which will be used to store the factorial result.
4. A while loop runs as long as 'n' is not equal to 0.
5. Inside the loop, 'fac' is multiplied by 'n' to calculate the factorial, and 'n' is decremented by 1.
6. Once the loop finishes, the program prints the calculated factorial stored in 'fac'.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This program calculates the factorial of a given number. It takes an integer input from the user, multiplies all integers from the input down to 1, and prints the result. The program uses a while loop to iterate through the numbers and calculate the factorial. The final output is the factorial of the input number.
