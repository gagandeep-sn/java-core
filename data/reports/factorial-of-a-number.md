# Factorial of a Number

## Category

Math

## Difficulty

Easy

## Concepts

* Recursion
* Loops

## Code Logic

1. The program starts by asking the user to enter a number, which is stored in the variable 'num'.
2. The 'factorial' method is called with 'num' as an argument.
3. Inside the 'factorial' method, a variable 'fac' is initialized with the value of 'n' (the input number).
4. If 'n' is 1, 'fac' is set to 1, since the factorial of 1 is 1.
5. A for loop is used to iterate from 'n-1' down to 1, multiplying 'fac' by each number in the sequence.
6. After the loop finishes, 'fac' holds the factorial of the original number, which is returned and printed to the console.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This program calculates the factorial of a given number. The user is prompted to enter a number, and the program calculates its factorial by multiplying all integers from the number down to 1. The result is then printed to the console.
