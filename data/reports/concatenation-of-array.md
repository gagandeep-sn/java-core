# Concatenation of Array

## Category

Arrays

## Difficulty

Easy

## Concepts

* Array manipulation
* Looping

## Code Logic

1. We first initialize an array 'nums' with the values [1, 2, 3, 4] and store its length in the variable 'n'.
2. We create a new array 'ans' with a length that is twice the length of 'nums'.
3. We use a for loop to iterate over the elements of 'nums' and copy each element to the corresponding index in 'ans' and also to the index that is 'n' positions ahead.
4. We use another for loop to print out the elements of 'ans'.

## Time Complexity

O(n)

## Space Complexity

O(n)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This program takes an array of integers as input and creates a new array that is a concatenation of the original array with itself. The program then prints out the elements of the new array. The input array is [1, 2, 3, 4] and the output is a new array where each element of the original array is repeated twice.
