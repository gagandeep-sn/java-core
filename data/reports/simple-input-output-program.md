# Simple Input Output Program

## Category

Input Output

## Difficulty

Easy

## Concepts

* Scanner
* Input Output

## Code Logic

1. The program creates a new Scanner object called 'in' to read input from the console.
2. The 'in.next()' method is used to read a single string input from the user and store it in the variable 'n'.
3. The program then prints the value of 'n' to the console using 'System.out.println(n)'.
4. The program does not perform any error checking or processing on the input, it simply reads and prints it.

## Time Complexity

O(1)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This Java program takes a single string input from the user and prints it back to the console. It uses the Scanner class to read the input. The program does not perform any processing on the input, it simply echoes it back. The output is the same string that the user entered.
