# Product of Two Numbers

## Category

Math

## Difficulty

Easy

## Concepts

* Input Output
* Basic Operations

## Code Logic

1. The program starts by asking the user to enter two numbers using a Scanner.
2. The input numbers are stored in the variables num1 and num2.
3. The program calls the product method, passing num1 and num2 as arguments.
4. Inside the product method, the product of num1 and num2 is calculated and stored in the variable product.
5. The product is then returned to the main method and printed to the console.

## Time Complexity

O(1)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This Java program takes two integers as input from the user, multiplies them together, and prints the product. The program uses a Scanner to read the input and a simple method to calculate the product. The result is then printed to the console.
