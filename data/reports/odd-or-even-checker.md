# Odd or Even Checker

## Category

Miscellaneous

## Difficulty

Easy

## Concepts

* Input Output
* Conditional Statements

## Code Logic

1. The program prompts the user to enter an integer and stores it in the variable 'n'.
2. The 'Check' method is called with the input integer 'n' as its argument.
3. Inside the 'Check' method, the modulus operator '%' is used to find the remainder of the division of 'q' by 2.
4. If the remainder is 0, the number is even, and the program prints a message stating that 'q' is even.
5. If the remainder is not 0, the number is odd, and the program prints a message stating that 'q' is odd.

## Time Complexity

O(1)

## Space Complexity

O(1)

## Explanation

This Java program checks whether a given integer is odd or even. It takes an integer input from the user, performs a simple calculation to determine its parity, and then prints the result. The program uses a conditional statement to make the determination. The output is a message stating whether the number is odd or even.
