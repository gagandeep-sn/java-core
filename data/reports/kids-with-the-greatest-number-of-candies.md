# Kids With the Greatest Number of Candies

## Category

Arrays

## Difficulty

Easy

## Concepts

* Arrays
* Loops

## Code Logic

1. Initialize an array to store the number of candies each kid has and the number of extra candies.
2. Initialize a variable to store the maximum number of candies any kid has.
3. Loop through the array to find the maximum number of candies any kid has.
4. Initialize a boolean array to store the result for each kid.
5. Loop through the array again, and for each kid, check if they can have the greatest number of candies if they are given the extra candies.
6. If the kid can have the greatest number of candies, set the corresponding value in the boolean array to true, otherwise set it to false.
7. Print the boolean array as the result.

## Time Complexity

O(n)

## Space Complexity

O(n)

## Interview Relevance

Medium

## Topic Mastery

Beginner

## Explanation

This program determines which kids can have the greatest number of candies if they are given extra candies. It takes an array of the number of candies each kid has and the number of extra candies as input. It then checks if each kid can have the greatest number of candies if they are given the extra candies. The output is a boolean array where true means the kid can have the greatest number of candies and false means they cannot.
