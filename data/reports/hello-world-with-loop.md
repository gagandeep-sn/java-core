# Hello World with Loop

## Category

Miscellaneous

## Difficulty

Easy

## Concepts

* Input Output
* Loops

## Code Logic

1. The program starts by printing a welcome message using System.out.printf.
2. A for loop is initialized with a counter variable i set to 1.
3. The loop condition is set to i <= 5, meaning the loop will run as long as i is less than or equal to 5.
4. Inside the loop, the current value of i is printed to the console using System.out.println.
5. After printing the value of i, the loop counter is incremented by 1.
6. Steps 3-5 repeat until the loop condition is no longer met, at which point the loop ends.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This Java program prints a welcome message and then uses a for loop to print the numbers 1 through 5. The loop iterates five times, printing the current value of the loop counter each time. The output is a series of lines with the welcome message followed by the numbers 1 through 5.
