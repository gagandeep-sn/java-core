## Code Logic

1. Iterate through the array or collection element by element
2. Compare each element against a condition inside the loop
3. Track the result (max, min, count, or a specific value) as you go
4. After the loop ends, return or print the final result
5. The main method ties everything together — it sets up test data and invokes the core logic
