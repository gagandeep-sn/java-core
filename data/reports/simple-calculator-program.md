# Simple Calculator Program

## Category

Miscellaneous

## Difficulty

Easy

## Concepts

* Switch Statement
* Input Output
* Basic Arithmetic Operations

## Code Logic

1. The program starts by importing the necessary Scanner class for input output operations.
2. It then creates a Scanner object to read input from the user.
3. The program prompts the user to enter the first number and stores it in the variable num1.
4. It then prompts the user to enter the second number and stores it in the variable num2.
5. The program asks the user to enter an operator (+, -, *, /, %) and stores it in the variable op.
6. A switch statement is used to perform different operations based on the operator entered by the user.
7. For each case in the switch statement, the program performs the corresponding arithmetic operation and prints the result.
8. If the user enters an invalid operator, the program prints an error message.
9. Finally, the program closes the Scanner object to prevent resource leaks.

## Explanation

This program is a simple calculator that takes two numbers and an operator as input from the user. It then performs the specified operation on the numbers and prints the result. The program handles basic arithmetic operations like addition, subtraction, multiplication, division, and modulus. It also checks for division by zero and invalid operators.
