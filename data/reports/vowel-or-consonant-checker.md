# Vowel or Consonant Checker

## Category

Strings

## Difficulty

Easy

## Concepts

* Character class
* Conditional statements

## Code Logic

1. The program prompts the user to enter an alphabet and stores it in the variable 'd'.
2. It converts the input character to lowercase using the 'Character.toLowerCase()' method and stores it in the variable 'c'.
3. It checks if the input character is a letter using the 'Character.isLetter()' method. If it's not a letter, it prints an error message.
4. If the input character is a letter, it checks if it's a vowel by comparing it with the vowels 'a', 'e', 'i', 'o', and 'u'.
5. If the input character is a vowel, it prints a message indicating that it's a vowel. Otherwise, it prints a message indicating that it's a consonant.

## Time Complexity

O(1)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This Java program checks whether a given alphabet is a vowel or a consonant. It takes a single character input from the user, converts it to lowercase, and checks if it's a letter. If it's a letter, it checks if it's a vowel or a consonant and prints the result.
