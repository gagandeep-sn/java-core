# Calculate CGPA

## Category

Miscellaneous

## Difficulty

Easy

## Concepts

* Input Output
* Basic Math Operations

## Code Logic

1. The program starts by asking the user to input the number of semesters they have completed.
2. It then initializes a variable 'total' to store the sum of all semester percentages.
3. A for loop is used to iterate over each semester, asking the user to input the percentage for that semester.
4. The input percentage is added to the 'total' variable in each iteration.
5. After the loop finishes, the overall percentage is calculated by dividing the 'total' by the number of semesters.
6. The overall percentage is then converted to CGPA by dividing it by 9.5.
7. Finally, the calculated CGPA is displayed to the user with two decimal places.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This program calculates the cumulative grade point average (CGPA) of a student based on their semester percentages. The user is asked to input the number of semesters and the percentage for each semester. The program then calculates the overall percentage and converts it to CGPA. The final CGPA is displayed to the user.
