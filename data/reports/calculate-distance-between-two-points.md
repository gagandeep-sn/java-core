# Calculate Distance Between Two Points

## Category

Math

## Difficulty

Easy

## Concepts

* Math.sqrt
* Math.pow

## Code Logic

1. The program prompts the user to enter the x and y coordinates of two points.
2. It uses a Scanner object to read the input values from the user.
3. The program calculates the difference between the x and y coordinates of the two points.
4. It uses the Math.pow function to square the differences and the Math.sqrt function to calculate the square root of the sum of the squared differences.
5. The result is stored in the 'dist' variable, which represents the distance between the two points.
6. Finally, the program prints the calculated distance.

## Time Complexity

O(1)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This program calculates the distance between two points in a 2D plane. It takes the x and y coordinates of two points as input, calculates the distance using the Euclidean distance formula, and prints the result. The program uses the Math class for square root and power operations.
