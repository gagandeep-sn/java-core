# Power Calculator

## Category

Math

## Difficulty

Easy

## Concepts

* Input Output
* Loops

## Code Logic

1. The program prompts the user to enter a number and stores it in variable 'a'.
2. It then prompts the user to enter the power and stores it in variable 'b'.
3. The program initializes a variable 'n' to 1, which will be used to store the result.
4. It enters a while loop that runs 'b' times, where 'b' is the power.
5. Inside the loop, it calculates the result by multiplying 'n' with 'a' and stores it back in 'n'.
6. After the loop finishes, it prints the final result, which is the power of the given number.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This Java program calculates the power of a given number. It takes two inputs from the user: the base number and the power. It then calculates the result by repeatedly multiplying the base number and prints the final result.
