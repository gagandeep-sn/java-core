# RichestCustomerWealth

## Category

Input Output

## Difficulty

Easy

## Concepts

* Scanner
* User Input

## Code Logic

1. Use nested loops to go through each row and column of the 2D array
2. The outer loop iterates over each row
3. The inner loop iterates over each column in the current row
4. Process or access each element at position [row][column]
5. Print the output directly to the console using System.out.println()
