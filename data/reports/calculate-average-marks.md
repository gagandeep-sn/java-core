# Calculate Average Marks

## Category

Miscellaneous

## Difficulty

Easy

## Concepts

* Input Output
* Basic Operations

## Code Logic

1. The program starts by importing the necessary Scanner class for input operations.
2. It then prompts the user to enter the number of subjects and stores the input in the 'subject' variable.
3. The program initializes a variable 'total' to store the total marks, which is initially set to 0.
4. A for loop runs from 1 to the number of subjects, prompting the user to enter the marks for each subject.
5. Inside the loop, the program reads the marks for each subject using 'i.nextDouble()' and adds it to the 'total' variable.
6. After the loop finishes, the program calculates the average marks by dividing the total marks by the number of subjects.
7. Finally, the program prints the average marks rounded to two decimal places using 'System.out.printf()'.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This Java program calculates the average marks of a student across multiple subjects. It takes the number of subjects and their respective marks as input, calculates the total marks, and then computes the average. The program outputs the average marks rounded to two decimal places.
