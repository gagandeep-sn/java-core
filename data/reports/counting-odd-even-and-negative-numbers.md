# Counting Odd, Even, and Negative Numbers

## Category

Input Output

## Difficulty

Easy

## Concepts

* Input/Output
* Conditional Statements
* Loops

## Code Logic

1. The program starts by creating a Scanner object to read input from the user.
2. It initializes three variables (negsum, evensum, oddsum) to store the sum of negative numbers, even numbers, and odd numbers respectively.
3. The program enters a while loop that continues indefinitely until the user enters 0.
4. Inside the loop, the program reads an integer from the user and checks if it's 0. If it is, the loop breaks and the program ends.
5. If the number is negative, it adds the number to the negsum variable.
6. If the number is even (i.e., the remainder when divided by 2 is 0), it adds the number to the evensum variable.
7. If the number is odd, it adds the number to the oddsum variable.
8. Once the loop ends, the program prints out the sums of negative numbers, even numbers, and odd numbers.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This program takes a series of integers as input from the user and calculates the sum of negative numbers, even numbers, and odd numbers separately. The input continues until the user enters 0. The program then prints out the sums of negative, even, and odd numbers.
