# Average of N Numbers

## Category

Input Output

## Difficulty

Easy

## Concepts

* Input/Output
* Loops
* Variables

## Code Logic

1. We create a Scanner object to read input from the user.
2. We initialize two variables, sum and count, to keep track of the sum of the numbers and the number of inputs, respectively.
3. We enter a loop that continues indefinitely until the user enters 0.
4. Inside the loop, we prompt the user to enter a number and read it using the Scanner object.
5. If the user enters 0, we break out of the loop.
6. Otherwise, we increment the count and add the number to the sum.
7. After the loop ends, we calculate the average by dividing the sum by the count.
8. Finally, we print the average to the console.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Interview Relevance

Low

## Topic Mastery

Beginner

## Explanation

This program calculates the average of a series of numbers input by the user. The user is prompted to enter numbers, and when they enter 0, the program stops taking input and calculates the average. The average is then printed to the console. The program uses a simple loop to accumulate the sum of the numbers and count the number of inputs.
